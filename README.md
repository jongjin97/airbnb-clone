# airbnb-clone
이 프로젝트는 AirBnb 웹 애플리케이션의 클론 버전을 만들었습니다. 사용자는 숙소를 검색하고 예약할 수 있으며, 호스트는 숙소를 등록하고 예약을 관리할 수 있습니다

# 프로젝트 구성
## Backend
- Spring Boot
- Spring Security
- Jwt
- Oauth2
- MongoDB
- Swagger
- WebSocket
## Frontend
- React
- TypeScript
- Node.js
- Redux
- Axios
- Tailwind CSS
- React Router
